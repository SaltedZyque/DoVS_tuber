# Run Instructions
1. Ensure Webcam is On
2. Install required modules (with pip install)
3. Run [face_tracker.py](face_tracker.py)

OpenCV
Matplotlib (optional)
Dlib (CMake and [Visual C++](https://visualstudio.microsoft.com/visual-cpp-build-tools/) install needed)
Numpy
Scipy
Imutils
